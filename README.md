# Real-Time Data Pipeline Project

## Overview  
This project aims to create a real-time data pipeline that efficiently processes streaming data from various sources and analyzes it in real-time. It will utilize tools and technologies such as Apache Kafka, Apache Spark, and various data storage solutions.

## Objectives  
- Stream data from multiple sources in real-time.
- Process and analyze the data efficiently.
- Store the results for further analysis.
- Provide an interface for querying the data.

## Technologies Used  
- **Data Streaming:** Apache Kafka  
- **Data Processing:** Apache Spark  
- **Data Storage:** MongoDB, PostgreSQL  
- **Containerization:** Docker  
- **Orchestration:** Kubernetes  

## Installation  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/Yannick-Maya/pipeline-journal.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd pipeline-journal  
   ```  
3. Build the Docker images:  
   ```bash  
   docker-compose build  
   ```  
4. Run the services:  
   ```bash  
   docker-compose up  
   ```
